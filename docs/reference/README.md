# Historical and Reference Docs

Indice de documentacion historica y de referencia.

Esta capa conserva memoria institucional y contexto estrategico o tecnico que sigue siendo util, pero no gobierna por si sola la ejecucion diaria.

## Uso correcto

Consulta esta capa cuando necesites:

- entender el origen del proyecto
- revisar el pivot estrategico
- recuperar razonamientos previos
- estudiar propuestas o evaluaciones
- contextualizar decisiones arquitectonicas

## Regla de prioridad

- prevalece la documentacion operativa
- la discrepancia debe resolverse actualizando la capa viva
