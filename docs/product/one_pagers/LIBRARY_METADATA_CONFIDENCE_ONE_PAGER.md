# Library Metadata Confidence One-Pager

## Para quien es

- biblioteca universitaria
- equipos de metadatos
- catalogacion con componente analitico

## Problema

La capa de registros sostiene muchas decisiones posteriores, pero hoy suele sufrir:

- variantes de autores e instituciones
- inconsistencias metadata
- baja trazabilidad de limpieza
- poca confianza para analitica downstream

## Lo que UKIP promete

`UKIP permite convertir un conjunto heterogeneo de registros en una capa más confiable para catalogo, analitica y reportes.`

## Lo que mas importa para esta audiencia

- control de autoridades
- armonizacion
- confianza en la capa de registros
- trazabilidad de curacion

## Lo que ve en el producto

- `Authority Review`
- `Harmonization Log`
- tabla con `Estado del sistema` y `Estado de revision`
- brief con stakeholder lens de biblioteca/metadatos

## Señal de valor temprana

El piloto funciona si demuestra que:

- mejora la consistencia del registro
- reduce ambiguedad visible
- deja mas claro donde conviene enfocar curacion

## Oferta sugerida

`Metadata Confidence Pilot`

Objetivo:

- demostrar mejora visible en confianza y consistencia del registro

## CTA sugerido

`Trabajemos un lote real de registros ambiguos y midamos cuánto mejora la confianza del dataset después de curacion y enrichment.`
