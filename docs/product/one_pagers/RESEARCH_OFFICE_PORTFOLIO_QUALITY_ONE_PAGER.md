# Research Office Portfolio Quality One-Pager

## Para quien es

- oficina de investigacion
- coordinacion de evaluacion
- equipos de seguimiento de portafolio

## Problema

La oficina necesita reportar y seguir el portafolio, pero pierde tiempo en:

- cruzar exportaciones
- limpiar registros
- corregir variantes
- explicar por que el dato aun no es confiable

## Lo que UKIP promete

`UKIP ayuda a la oficina de investigacion a pasar de datasets dispersos a una base operativa más confiable para seguimiento y reporte institucional.`

## Lo que mas importa para esta audiencia

- cobertura de enrichment
- calidad promedio
- brechas benchmark
- acciones operativas claras

## Lo que ve en el producto

- tabla de conocimiento mas alineada con filtros, facets y estados
- authority review y calidad/revision diferenciadas
- benchmark con reglas configurables
- scientific import validado con smoke test real

## Señal de valor temprana

El piloto funciona si demuestra que:

- mejora la confianza operacional del dataset
- se vuelven visibles las brechas reales
- ya existe una linea base reutilizable para seguimiento

## Oferta sugerida

`Portfolio Quality Pilot`

Objetivo:

- elevar confianza del dataset y dejar una linea base operativa para reporte institucional

## CTA sugerido

`Probemos con una muestra real y midamos cobertura, calidad y gaps operativos antes de ampliar el alcance.`
