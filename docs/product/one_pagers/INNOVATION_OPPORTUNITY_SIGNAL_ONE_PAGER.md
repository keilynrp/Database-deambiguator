# Innovation Opportunity Signal One-Pager

## Para quien es

- innovacion
- transferencia
- scouting institucional
- vinculacion con socios

## Problema

El portafolio contiene señales valiosas, pero cuesta mucho detectar:

- outputs destacados
- areas de concentracion
- señales de impacto
- oportunidades a explorar rapido

## Lo que UKIP promete

`UKIP ayuda a convertir el portafolio en una lectura más rápida de outputs destacados, concentraciones tematicas y oportunidades a explorar.`

## Lo que mas importa para esta audiencia

- top entities por impacto
- clusters conceptuales
- señales destacables
- narrativa breve para opportunity scanning

## Lo que ve en el producto

- `Top Entities by Impact`
- `Knowledge Concept Map`
- brief con stakeholder lens de innovacion / transferencia

## Señal de valor temprana

El piloto funciona si demuestra que:

- aparecen outputs visibles para seguimiento
- emergen areas de concentracion que vale la pena explorar
- se vuelve mas clara una ruta exploratoria de oportunidad

## Oferta sugerida

`Opportunity Signal Pilot`

Objetivo:

- validar si el portafolio ya produce señales utiles para exploracion, vinculacion o seguimiento

## CTA sugerido

`Tomemos un subconjunto del portafolio y evaluemos si aparecen outputs y concentraciones que justifiquen seguimiento u oportunidad.`
