# US-015 - Backfillear el frente de automation y delivery

## 1. User story

Como equipo de producto, quiero formalizar automation and delivery como frente operativo para poder priorizar reportes, alertas, imports programados y workflows sin perder cohesion.

## 2. Context

Estas capacidades existen y agregan mucho valor, pero faltaba darles identidad de backlog dentro del sistema operativo nuevo.
